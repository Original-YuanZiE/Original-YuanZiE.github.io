系统安装教程


1.安装前的准备

*请提前备份好所有数据！*

确认你的电脑是否满足最低配置要求：
·CPU：6th Gen Intel Core i5 或其它同性能 CPU
·RAM：4GB DDR4
·存储：剩余至少 80GB 的空间
·GPU：Intel UHD Graphics 或其它核显
·屏幕：1920 × 1080 @ 30Hz
·操作方式：键鼠，可选触摸

你需要一个 Windows PE 来辅助 see-sawOS 的安装，请准备一个 8GB 以上的可移动存储设备，并备份好数据，前往以下链接下载 FirPE： 下载 FirPE *再提醒一次，备份好可移动存储设备的数据！*
按照网页上的教程制作好 Windows PE 启动盘后，便可以正式安装系统了
2.安装系统

将电脑关机，插入 Windows PE 启动盘，按下电源键开机并狂按进入 UEFI BIOS 的快捷键（一般为 F2 或 DEL，具体可以在网上搜索：品牌 + “进入 BIOS”），将第一启动项设为启动盘（具体设置方法也可以在网上查询），保存并退出，等待 Windows PE 启动
